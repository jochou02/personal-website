eSports Data Modeling [JavaScript, Python (Flask, Pandas, BeautifulSoup)]

  • Developed a Python-based predictive model for professional League of Legends matches, designed to provide data-driven insights for optimized betting strategies.
  • Utilized web scraping techniques to extract data from online databases, with a comprehensive dataset of over 4,000 professional matches and 2,500 professional players.
  • Implemented the initial model on a Jupyter Notebook, using various Python libraries for data extraction, cleaning, and analysis.
  • Applied backtesting to iteratively refine and improve the model, achieving an accuracy rate of 79.6%, based on expected outcomes as per sportsbook odds.
  • Transformed the Jupyter Notebook model into a full-stack web application using Flask, making it accessible via an API and enhancing its usability.
  • Hosted the application backend on Render.com, facilitating continuous iterative improvement of model performance through accessible backtesting.

Auto Messenger

I thought of this idea while texting my family in Taiwan over Line, while constantly swapping to Google Translate ...

Navigating the language barrier became a constant challenge, as they couldn't understand English and I was unable to read Chinese. This messaging app I'm working on is designed to surmount such language barriers. By leveraging automatic translation, it instantly converts incoming messages into the recipient's preferred language, fostering seamless interaction regardless of linguistic differences.
